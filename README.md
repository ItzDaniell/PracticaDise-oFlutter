# Practica Diseño Flutter

Proyecto Flutter de práctica para diseño de interfaces y componentes.

## Requisitos

- Flutter SDK 3.x
- Dart 3.x

## Instalación y configuración

1. Ejecuta la app:

```bash
flutter run
```

## Estructura básica

- `lib/`: código fuente principal
- `lib/main.dart`: punto de entrada de la app
